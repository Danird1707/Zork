# Zork

## Description

Zork is a text adventure game written in C++. The player explores several rooms, collects items, opens containers, solves a small puzzle and unlocks the treasure room.

## How to Play

Run the executable and type commands in the console.

Available commands:

- `look`
- `go north`
- `go south`
- `go east`
- `go west`
- `take item`
- `drop item`
- `inventory`
- `open item`
- `read item`
- `use item on target`
- `unlock direction with key`
- `quit`

## Walkthrough

1. Start in the Hall.
2. Use `open box`.
3. Use `take note`.
4. Use `read note`.
5. Go to the Kitchen with `go north`.
6. Take the lever with `take lever`.
7. Return to the Hall with `go south`.
8. Go to the Library with `go east`.
9. Use `use lever on lockbox`.
10. Take the special key with `take special_key`.
11. Go to the fourth room.
12. Use `unlock east with special_key`.
13. Use `go east`.
14. Open or read the final letter to complete the game.

## Goal

Find the special key, unlock the treasure room and read the final congratulation letter.
